📖GangChew 개요 ​

본 프로젝트는 한국소프트웨어기술진흥협회(KOSTA) 에서 진행한 Java 기반 DevOps 개발자 양성 훈련과정 중 최종프로젝트로 작성되었습니다. 어디 이런 강의 없을까? 라는 물음에서 시작된 GangChew는 사용자의 니즈가 반영된 강의를 제공하는것을 목표로 합니다.

사용자가 자유롭게 강의를 요구하는 추천 게시판과 실제 강의 계획서를 작성해 펀딩을 모집하는 펀딩 게시판으로 이루어져있습니다.

2. 프로젝트 주요 기능 ​

강의 추천 게시판 ​

  • 사용자가 원하는 강의를 자유롭게 요청
  • 추천 수와 댓글을 통해 수요 확인 가능

펀딩 게시판 ​

  • 사용자가 직접 강의 개설 및 펀딩 모집 가능
  • 일정 기간 내 목표 금액 도달 시 강의 개설
  • 기간 초과 또는 강의 개설자의 취소 시 자동 환불

외부 API 연동 ​

  • 카카오 주소 API : 회원 가입/펀딩 강의 위치 등록
  • 카카오/네이버 로그인 API : 간편 회원 가입 및 로그인
  • 아임포트 결제 모듈 : 신용카드, 모바일 결제(다날), 카카오페이, 토스페이 지원

3. 기술 스택 ​

  • Backend : Java, Spring Boot, JPA
  • Frontend : React.js
  • Database : Oracle
