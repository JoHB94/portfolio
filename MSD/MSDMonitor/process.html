👀MSD 생산 모니터링 ​

생산 공정에서 생산된 제품을 실시간으로 모니터링하는 시스템을 도입하여 공정 효율성을 극대화합니다. 또한, 장비 및 센서에서 발생하는 데이터를 실시간으로 수집·통신하여 공정 상황을 직관적으로 파악할 수 있도록 지원합니다. 이를 통해 빠른 의사결정과 공정 최적화를 실현할 수 있습니다.

1️⃣ 시연 영상 ​

2️⃣ 주요 화면 및 기능 ​

🖥️ 생산 공정 실시간 모니터링 ​

📌 기능 설명 ​

  1. 생산 공정 실시간 모니터링
    • 각 공정 단계의 진행 상태를 실시간으로 확인할 수 있는 대시보드 제공.
    • 센서 데이터, 장비 상태, 작업 진행 상황을 시각화하여 직관적인 모니터링 환경 구축.
  2. MQTT 기반 데이터 통신
    • RabbitMQ 메시지 브로커를 활용하여 실시간 데이터의 발행(Publish) 및 구독(Subscribe) 기능 구현.
    • 데이터 전송 지연을 최소화하여 생산 라인의 신속한 의사결정을 지원하고, 실시간 이벤트 기반 대응 가능.

🏭 생산 계획 대시보드 ​

📌 기능 설명 ​

  1. 생산계획 / 지시
    • 제품명을 입력하면 해당 연도의 총 생산 계획을 확인할 수 있는 기능 제공.
    • 연간 생산 계획을 1주차부터 52주차까지 주별로 설정 가능.
  2. 일일계획 / 지시
    • 현재 날짜를 기준으로 해당 주차의 저장된 제품명 및 주별 생산량을 확인하고, 월요일부터 일요일까지의 일일 생산 계획을 수립 가능.
    • 특정 날짜를 선택하여 다른 주차의 생산 계획을 조회 및 수정 가능.
  3. 서버 연결(서버내용넣을것임)
    • API를 통해 서버와 연동하여 생산 계획 데이터를 실시간으로 송수신 및 업데이트 가능.
