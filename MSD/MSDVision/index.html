🎥품질 검사 AI - MSDVision ​

🔍MSDVision이란? ​

MSDVision은 스마트 팩토리에서 AI 기반 비전 검사 시스템을 활용하여 제품 품질을 자동으로 분석하고 최적화하는 솔루션으로,
기존의 육안 검사보다 빠르고 정확한 불량 감지, 품질 데이터 수집, 실시간 피드백을 제공하여 제조 공정을 효율적으로 개선합니다.

핵심 기능 ​

  • 실시간 비전 검사: AI 딥러닝 모델 기반의 실시간 품질 검사
  • 불량 탐지 & 분류: 제품의 미세한 결함까지 정밀하게 식별
  • 자동 데이터 로깅: 검사 결과를 MES/SCADA 시스템과 연계하여 품질 관리 자동화
  • 실시간 분석 & 피드백: 검사 데이터를 활용한 품질 개선 피드백 제공

🎯해결하는 문제 ​

🚨 문제점✅ MSDVision 솔루션
육안 검사로 인한 오류AI 모델을 활용한 자동 품질 분석
작업자의 피로도 증가반복 작업을 자동화하여 효율성 향상
불량률 증가실시간 데이터 분석을 통한 품질 개선
수작업 데이터 기록MES/SCADA 연계를 통한 자동 데이터 로깅

🛠️핵심 기술 ​

MSDVision은 AI 기반 이미지 처리 기술과 실시간 데이터 연계 시스템을 결합하여 높은 정확도를 보장합니다.

Vision AI 기반 품질 검사 ​

  • ONNX Runtime을 활용한 경량 AI 모델 서빙
  • OpenCV 기반의 이미지 전처리 및 분석

실시간 데이터 처리 & 통신 ​

  • 검사 데이터는 MES/SCADA 시스템과 연계하여 품질 이력 관리
  • TCP 소켓 및 MQTT 기반의 실시간 데이터 전송
