📡 실시간 데이터 처리 및 통신 ​

MSDVision은 품질 검사 데이터를 MES/SCADA 및 데이터베이스에 실시간으로 전송 및 저장하기 위해 MQTT, TCP 소켓, HTTP 통신을 병행하여 사용합니다.
각 통신 방식은 특정 목적에 맞게 최적화되어 있으며, 효율적인 데이터 전송과 저장을 보장합니다.

1️⃣ MQTT를 활용한 MES/SCADA 실시간 검사 현황 전송 ​

시연 영상 ​

🔹 MQTT를 사용하는 이유 ​

  • 경량 프로토콜: MQTT는 저전력, 저대역폭 환경에서도 안정적인 데이터 전송 가능
  • Stage 단위 데이터 전송: 센서 기반 검사 단계별 데이터를 실시간으로 송신
  • SCADA 및 MES 연동 최적화: 스마트 팩토리 내 SCADA 시스템과의 원활한 연동 지원

📌 MQTT 데이터 전송 구조 ​

단계설명
센서 데이터 수집카메라 및 센서에서 제품 검사 현황 수집
MQTT Broker로 전송RabbitMQ MQTT 브로커 사용
MES/SCADA 구독SCADA 및 MES 시스템이 Subscriber 역할 수행
실시간 현황 모니터링SCADA 화면에서 품질 검사 현황을 실시간 확인 가능

2️⃣ TCP 소켓을 활용한 대량 검사 결과 데이터 저장 (제품 단위) ​

📹 시연 영상 ​

🔹 TCP 소켓을 사용하는 이유 ​

  • 고속 전송 및 안정성: REST API보다 훨씬 빠르고, 대량 데이터 전송 시 지연이 적음
  • 시계열 데이터 최적화: 하루 수십만 건의 검사 데이터를 시계열 DB (Time-Series DB)에 저장하는데 최적화
  • 연결 지향적 프로토콜: 데이터 손실 없이 안정적인 저장 가능
