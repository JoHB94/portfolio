📝 요구사항 ​

🔑 로그인 ​

  • 고객은 이름과 휴대폰 번호를 입력하여 로그인합니다.

📌 고객 여행 견적 신청 ​

  1. 고객명 & 휴대폰번호 → 로그인한 유저 정보를 자동 출력
  2. 여행 기간 → 1~30일 사이로 선택 가능
  3. 이동수단 선택 → 렌트(R), 대중교통(B), 자차(C)
    • 렌트(R)를 선택하지 않으면 상세 교통편에서 렌트 선택 불가
  4. 여행지 선택 → 전국 시/도 선택 가능

📅 고객 여행 일정 관리 ​

  1. 고객 리스트 조회
    • 여행 견적을 신청한 고객 리스트 출력
    • 고객명을 클릭하면 하단에 여행 일정 스케줄 출력
  2. 날짜 버튼 생성
    • 선택한 고객의 여행 기간에 따라 일별 버튼 자동 생성
    • 버튼 클릭 시 해당 날짜의 상세 일정 입력 폼 출력
  3. 견적 경비 강조 표시
    • 예상 경비 초과 시 견적 경비를 빨간색으로 표시

🛠️ 고객 여행 견적 신청 - 일정 추가 및 수정 ​

  1. 추가/삭제 버튼
    • 일정 행을 추가할 수 있으며, 체크박스 선택 후 삭제 가능
  2. 상세 일정 입력
    • 시간 중복 방지:
      • 스케줄 등록 시 겹치는 시간(예상 이동 시간 + 이용 시간) 선택 불가
      • 스케줄 가능 시간: 07시 ~ 익일 오전 04시
  3. 지역 선택
    • 첫 번째 드롭박스 → 고객이 여행 견적 신청에서 등록한 시/도 값 고정
    • 두 번째 드롭박스 → 선택한 시/도에 따라 시/군/구 행정구역 동적 설정
  4. 이동 수단 선택
    • 도보(W), 버스(B), 지하철(S), 택시(T), 렌트(R), 자차(C)
  5. 교통비 자동 계산
    • 각 행의 교통비는 10분 단위로 자동 산출

💰 교통비 계산 로직 ​

🚍 대중교통 (버스/지하철) ​

  • 기본 요금
    • 버스: 1,400원
    • 지하철: 1,450원
  • 추가 요금
    • 20분 초과 시 → 20분당 200원 할증

🚖 택시 ​

  • 기본 요금: 3,800원
  • 거리 별 할증: 이동 거리 10분당 5,000원 추가
  • 시간 별 할증:
    • 22:00 ~ 24:00 → 20% 추가 요금
    • 24:00 ~ 04:00 → 40% 추가 요금

🚗 렌트카 ​

  • 렌트 비용:
    • 1~2일: 하루 10만 원
    • 3일 이상: 일 1만 원씩 할인 적용
  • 기름값: 이동 거리 10분당 500원 연료비 추가

🔄 여행 견적 확인 및 수정 요청 ​

  1. 고객 재로그인 시 → 신청한 견적 상세 내역 출력
  2. 등록된 여행 일정 리스트 확인 & 수정 요청 가능
    • 일정 수정 요청 (체크박스 선택 후 요청 버튼 클릭)
    • 수정 요청 상태 값:
      • M (수정 요청), C (수정 완료)
  3. 관리자 측에서 수정 요청 건 표시
    • 일정 수정 후 M → C 값 변경
