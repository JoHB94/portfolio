📖SI프로젝트 개요 ​

본 프로젝트는 국민취업지원제도의 인턴형 일경험 과정을 통해 수행한 Web SI 개인 프로젝트입니다. 실제 SI 회사에서 교육받은 내용을 바탕으로 개발하였으며, 요구사항 정의서에 맞춰 정확한 기능 구현을 목표로 삼았습니다.

특히, 본 교육 과정에서는 UI/UX 디자인 요소가 포함되지 않았으며, HTML table을 활용한 기본적인 레이아웃으로만 화면을 구성했습니다. 이로 인해 시각적인 완성도가 낮고, 뷰가 단순한 구조로 되어 있습니다.

📌 본 프로젝트를 통해 배운 점: ​

  • SI 개발 환경에서 요구사항을 정확히 반영하는 프로세스 경험
  • 디자인보다는 기능 구현과 요구사항 충족을 최우선으로 한 개발 경험
  • 전자정부 프레임 워크 기반의 개발 경험
  • 나의 생각대로 개발하는 것과 요구사항대로 개발하는것이 매우 다르다는것을 배움

기술 스택 ​

  • 언어 및 프레임워크: Java, SpringFrameWork, JQuery, Ajax, Mybatis
  • 데이터 베이스: Oracle
