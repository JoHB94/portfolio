📝 요구사항 ​

🔑 로그인 ​

  1. 이름과 휴대폰 번호를 입력하여 로그인합니다.

📌 입사지원 입력 ​

  1. 학력, 경력, 자격증 각각 입력 항목 추가/삭제 가능 (체크박스 선택 시)
  2. 필수 항목 입력 여부 검증 후 저장 및 제출 가능
    • 저장 → 재로그인 시 수정 가능
    • 제출 → 재로그인 시 수정 불가능
  3. 기본 인적사항과 학력사항은 필수 입력 요소
  4. 경력사항과 자격증명은 선택 입력 요소

📅 날짜 규칙 ​

  1. 공통 규칙

    • 1, 3, 5, 7, 8, 10, 12월 → 최대 31일까지 입력 가능
    • 4, 6, 9, 11월 → 최대 30일까지 입력 가능
    • 2월 → 윤달을 반영하여 28일 또는 29일까지 입력 가능
    • 재학 기간, 근무 기간, 취득 일자 → 생년월일 이후여야 함

  2. 재학 기간 규칙

    • 재학 시작일은 종료일보다 빨라야 한다.
    • 재학 기간은 서로 겹칠 수 없음

  3. 근무 기간 규칙

    • 근무 시작일은 종료일보다 빨라야 한다.
    • 근무 기간은 서로 겹칠 수 없음
    • 재학 기간과 근무 기간 또한 서로 겹칠 수 없음

🛠️ 입사지원 수정 ​

  1. 최초 등록 후 재로그인 시 변경사항

    • 학력사항, 경력사항, 희망연봉 → 회사 내규로 고정
    • 희망근무지/근무형태 테이블 추가

  2. 학력사항 표기 방식

    • 재학 기간이 4년 이상이면 대학교(4년)으로 표기
    • 재학 기간이 4년 미만이면 대학교(2년)으로 표기
    • 졸업/재학/중퇴 여부는 기간에 따른 구분 값으로 표시
    • 표기 우선순위:
      1. 4년/2년 여부
      2. 졸업 → 재학 → 중퇴 순서

  3. 경력사항 표시

    • 총 근무 기간 합산

  4. 입사지원 상태 값

    • 저장(Saved) 상태 → 수정 가능
    • 제출(Submitted) 상태 → 수정 불가능

  5. Submitted 상태에서는

    • 어떠한 입력도 불가능
    • 저장/제출 버튼이 비활성화됨
