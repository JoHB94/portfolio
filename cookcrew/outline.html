📖CookCrew 개요 ​

본 프로젝트는 한국품질재단(KFQ) 에서 진행한 빅데이터 분석 서비스 플랫폼 프론트엔드 개발자과정 중 최종프로젝트로 작성되었습니다. 다이어트를 어떻게 하면 맛있고 건강하게 할 수 있을까? 라는 물음에서 시작된 CookCrew는 사용자끼리 다이어트 레시피를 공유하고 식단 관리기능을 제공합니다. 또한 다양한 랭킹선정을 통해 사용자의 다이어트에 대한 동기를 부여합니다

2. 프로젝트 주요 기능 ​

다이어트 레시피 게시판 ​

  • 사용자가 자신의 레시피를 작성
  • 좋아요, 조회수 등 인기 레시피 선정

식단 관리 캘린더 ​

  • FullCalendarAPI를 사용하여 날짜 별로 목표 칼로리에 맞춰 식단을 등록
  • 계획 달성 여부에 의한 랭킹 산정을 통해 사용자에게 동기부여

공공 데이터 포털 사용 ​

  • 식품 의약품 안전처에서 제공하는 칼로리 정보가 포함된 레시피를 기반으로 다수의 다이어트 레시피들을 페이지에서 제공

3. 기술 스택 ​

  • Backend : Java, Spring Boot, JPA
  • Frontend : React.js
  • Database : MySQL
